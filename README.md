# 有幸的个人网站

这是一个简洁优雅的个人网站项目，采用纯HTML/CSS/JavaScript实现，通过CDN引入所需资源。

## 项目结构

```
.
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── script.js      # 交互脚本
└── README.md          # 项目说明
```

## 技术特点

- 纯HTML/CSS/JavaScript实现
- 响应式设计，支持移动端和桌面端
- 使用CDN引入所需资源
- 模块化文章展示
- 左右滑动交互
- 简洁优雅的UI设计

## 使用说明

直接在浏览器中打开 index.html 即可查看网站。
